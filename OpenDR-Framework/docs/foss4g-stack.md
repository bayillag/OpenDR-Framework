# FOSS4G Stack Alignment

OpenDR 1.0 is built 100% on OSGeo-aligned software to ensure technical transparency and verifiability.

| Project | Version | Role |
| :--- | :--- | :--- |
| **PostGIS** | 3.4 | Vector management and spatial joins |
| **PostgreSQL** | 16 | Relational database backend |
| **pygeoapi** | 0.15+ | OGC API implementation |
| **Dask-Geo** | 2024.x | Distributed geospatial compute |
| **GDAL/OGR** | 3.8.x | Data abstraction and ETL |
| **Apache Sedona** | 1.5 | 3D CityGML and big geometry processing |
| **QGIS** | 3.34 LTR | Expert visualization and analysis |
| **KoboToolbox** | Latest | Field data collection and validation |
| **Docker** | 25.x | Microservice containerization |
